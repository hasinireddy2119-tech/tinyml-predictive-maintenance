# TinyML Predictive Maintenance Node

## Overview

TinyML Predictive Maintenance Node is an ESP32-based edge AI system designed to detect machine anomalies in real time using vibration sensor data. The project leverages TensorFlow Lite for Microcontrollers to perform on-device inference, enabling low-latency and low-power predictive maintenance.

## Features

* Real-time vibration monitoring
* TinyML inference on ESP32
* Machine anomaly detection
* Edge processing without cloud dependency
* Low-power embedded deployment

## Hardware

* ESP32 Development Board
* MPU6050 Accelerometer/Gyroscope
* USB Power Supply

## Software

* Arduino IDE
* TensorFlow Lite for Microcontrollers
* C/C++
* Python (Model Training)
